Multi-Wavelength Galaxy Illustrator (MGI)

Complete User Manual and Software Overview

Introduction

The Multi-Wavelength Galaxy Illustrator (MGI) is a software framework that provides tools for knowledge discovery and exploration of astronomical FITS files, emphasizing multi-wavelength visualization of galaxies across desktop, VR, and AR platforms. It enables immersive data interaction and scientific visualization with the support of C# and Unity technologies.

MGI Versions

  • MGI-Desktop: 2D/3D FITS file visualizer for Windows.
  • MGI-VR: Fully immersive 3D galaxy exploration via VR headsets such as Oculus Quest, HTC Vive, or Valve Index.
  • MGI-AR: Educational and research AR implementation developed for the University of Portsmouth.

VR Specifications

  • Minimum 90Hz frame rate, prototype tested at 120Hz.
  • Uses OpenXR for cross-platform support.

Key Features and Tools

  • FITS File Visualization Interface
  • Image Manipulation (e.g., color mapping)
  • Interactive Tools (real-time navigation, scalability)
  • User-Generated Content support

System Requirements

MGI-Desktop: Windows 10+, Intel i5, 8GB RAM, GTX 1050+

MGI-VR: Windows 10+, Intel i7, 16GB RAM, GTX 1070+

MGI-AR: Windows Holographic, HoloLens, 16GB RAM (32GB recommended)

Installation and Configuration

Installation files and instructions are available at MGI GitHub Repository. Use Unity 2021.3.xf1 and CMake to build from source. OpenXR is required for VR configuration.

Setup for Unity & OpenXR

  1. Enable XR Plugin Management, OpenXR Plugin, Input System, and XR Interaction Toolkit in Unity.
  2. Enable OpenXR and configure interaction profiles in project settings.
  3. Create and bind XR input actions for controllers (grip, trigger, etc.).
  4. Set build target to Windows and enable OpenXR.

Configuration File Options

  • Colourisation Mode
  • Spatial Scaling
  • Cropping and Denoising of Galaxies

Workflows

  • Workflow 1: Basic Visualization for easy data access.
  • Workflow 2: Prototyping visualizations with parameter tuning.
  • Workflow 3: Repurposing the framework for custom tools.

Troubleshooting & Optimization

Ensure dependencies are installed. Lower graphics settings and update GPU drivers for performance. See FAQ for common issues.
