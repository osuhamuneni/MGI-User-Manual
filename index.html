Multi-Wavelength Galaxy Illustrator (MGI)

User Manual and Software Overview

Introduction

The Multi-Wavelength Galaxy Illustrator (MGI) is a software framework for visualizing astronomical FITS files across multiple wavelengths. It provides tools for desktop, virtual reality (VR), and augmented reality (AR) platforms.

MGI Versions

  • MGI-Desktop: Interactive FITS visualization in 2D/3D on desktop.
  • MGI-VR: Immersive 3D experience using VR headsets (Oculus, Vive, etc.).
  • MGI-AR: Augmented reality version for educational/research use.

Key Features

  • FITS file visualization
  • Interactive image manipulation
  • Real-time tools for immersive data exploration
  • User-generated content support
